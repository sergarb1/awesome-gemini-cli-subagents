# Awesome Gemini CLI Programming Agents 

## What is this?

This repository is a curated collection of **Gemini CLI subagents** specifically designed for **programming, software engineering, and technical development tasks**. 

The repository is structured to be "Plug & Play". Simply clone it or copy the agents you need.

## üöÄ Installation & Usage (Plug & Play)

The easiest way to use these agents is to clone this repository directly into your project's `.gemini` folder:

```bash
# Inside your project's root directory
git clone https://github.com/sergarb1/awesome-gemini-cli-subagents .gemini
```

Gemini CLI will automatically detect all agents in the `agents/` folder.

If you only want specific agents, you can just copy the `.md` files from the [**agents/**](agents/) directory to your project's `.gemini/agents/` folder.
